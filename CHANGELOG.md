# 更新日志

## v0.1.1 — 2026-07-13

### 新增

- 关于页新增“检查更新”入口；发现新版后可直接打开油猴更新确认页。
- 增加 `@updateURL` / `@downloadURL` 标准更新元数据，支持 Tampermonkey / Violentmonkey 检查更新。

### 优化

- 下载目录按“作者昵称 / 单条微博 ID”分层整理，例如 `weibo/大魔王FCP/R8eKMfiLR/`。
- 图片、普通视频及 Live Photo 视频统一保存到所属微博目录。
- 下载失败时不再回退到浏览器下载根目录，避免视频或图片散落在 `Downloads` 下。
- 媒体下载入口统一为微博卡片底栏“下载”按钮。
- 关于页导入 / 导出区域只保留“导出配置”和“导入配置”。

## v0.1.0 — 2026-07-13

- 首次公开发布。
- 提供阅读放大、界面清理、信息流过滤、收藏与媒体下载。
- 默认打开“最新微博”，支持卡片 IP 属地显示。
